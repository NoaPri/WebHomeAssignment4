# College Admissions Calculator - Assignment 4

### Description
A web-based calculator that checks university admission eligibility. The project features dynamic forms and validation based on age and academic background.

### Features
* **Input Toggling**: Automatically enables/disables fields based on the chosen track (Psychometric or Mechina).
* **Validation**: Checks for missing Bagrut grades.
* **Special Eligibility**: Candidates over 30 receive a specific notification with a green background.

### Technologies
* HTML5
* CSS3
* JavaScript

### Files
* `index.html` - The main structure.
* `style.css` - Styling and the `.green` class.
* `script.js` - Logic and functionality.

### How to Run
Open `college_admissions.html` in any web browser.
